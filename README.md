# Recoil을 이용한 TodoList 만들기

## Tech

React
TypeScript
Rocoil

### 프로젝트 생성

```
npx create-react-app ./ --template typescript
```

### 프로젝트 실행

```
npm start
```

### recoil 설치

```
npm install recoil
```

### axios 설치

```
npm install axios
```

### 라이브러리 설치

```
 npm i @emotion/styled @emotion/react react-icons uui
d
```
